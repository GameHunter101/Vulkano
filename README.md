## What is it
- Vulkan-powered graphics system
- Uses the rust Ash library

## What it does
- Text
- Images
- Rasterization
- Shaders
- 3D geometry
- Lighting
- Screenshots (saved as ./screenshot.jpg)
- Free movement
- Camera controls

## What the shaders do
- Ray marching
- Ambient occlusion
- Anti-aliasing
- Global illumination (togglable)
- Realistic lighting
